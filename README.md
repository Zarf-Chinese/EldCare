# 从数据库到后端代码的Auto-Generate
进入test/java/com.eldcare/CodeGenerator.java 文件并直接运行，按照提示来输入需要更新的表名，即可自动为这些数据表创建一系列代码。
# Ideal中部署项目
进入main/java/com.eldcare/Application.java文件并直接运行，即可在Ideal内启动后端
